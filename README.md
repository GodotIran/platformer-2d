# 🎮 نام بازی (تعیین نشده)

> پلتفرمر ۲بعدی متن‌باز با آرت استایل وکتور — ساخته‌شده با Godot 4

یک پروژه تیمی open-source فارسی‌زبان که هدفش ساخت یک بازی پلتفرمر ۲بعدی با ترکیب گیم‌پلی اکشن، پازل و ماجراجویی در دنیایی فانتزی است.

---

## ✅ وضعیت پروژه

| المان | تصمیم |
|-------|--------|
| ژانر | پلتفرمر ۲بعدی |
| آرت استایل | وکتور (Flat Vector) |
| موتور | Godot 4 / GDScript |
| پلتفرم | PC · Web · Mobile |
| تعداد بازیکن | تک‌نفره + Co-op آنلاین اختیاری |
| مکانیک | پلتفرمر + پازل محیطی + اکشن |
| حرکت | دوبل‌جامپ · داش · دیوارپریدن |
| سیستم HP | نوار سلامتی (چند ضربه) |
| سیستم مرگ | برگشت به آخرین Checkpoint |
| تم | فانتزی / ماجراجویی |

### پیش‌نیازها

- [Gdscript Formatter](https://github.com/GDQuest/GDScript-formatter)
 <details>

<p>گزینه Install or Update Formatter را بزنید</p>
<p>منتظر بمانید تا نصب کامل شود</p>
</details>
  
  
- یک اکانت GitHub
